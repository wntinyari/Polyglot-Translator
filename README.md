## What is PolyGlot?

PolyGlot is a free, web-based language translator that works instantly in any browser. It supports 50+ languages, features text-to-speech, and requires no downloads or accounts. Built with HTML, CSS, and JavaScript using the LibreTranslate API.**. 
It's:
- **Free** - No payments ever
- **Private** - No tracking, no data collection
- **Universal** - Works on any device
- **Offline** - Works without internet

## Key Features

1. **200+ Languages** - From Spanish to Swahili, including endangered languages
2. **Text-to-Speech** - Hear pronunciation of translated text
3. **Camera Translation** - Translate signs and menus with your camera
4. **Offline Mode** - Works without internet connection
5. **Privacy Focused** - Your translations stay on your device
